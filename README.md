# Explicación de lo entregado:
Se presentan dos repositorios correspondientes a esta actividad debido a una diferencia entre las indicaciones brindadas en clase y los requisitos detallados en la consigna de entrega.
El primer repositorio fue desarrollado utilizando MySQL, en base a lo explicado de forma oral durante la clase del lunes 30/3.
Posteriormente, al leer con mayor detalle la consigna publicada, se identificó que debía utilizarse SQLite. Por esta razón, se realizó nuevamente la actividad en un nuevo repositorio, cumpliendo con las especificaciones solicitadas.
Por eso, se adjuntan ambos repositorios, aclarando que el segundo corresponde a la versión final ajustada a la consigna.
